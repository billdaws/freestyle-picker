# freestyle-picker

For the indecisive soda drinker.

## Usage

```
$ ./freestyle.py -h
usage: freestyle.py [-h] [--diet] [--list]

Pick a random Coca-Cola Freestyle beverage.

options:
  -h, --help  show this help message and exit
  --diet      Pick only from diet / zero sugar / low calorie options.
  --list      List all available beverages and exit.
```
